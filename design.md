# CLOOVIES — UX/UI DESIGN SYSTEM

## Vision Produit

Cloovies n'est pas une application de gestion de films.

Cloovies est un rituel social.

L'application doit évoquer :

* L'attente
* La curiosité
* Le cinéma indépendant
* Les clubs privés
* Les conversations de fin de soirée
* Les émotions cachées qui seront révélées

L'utilisateur ne doit jamais avoir l'impression d'utiliser un outil.

Il doit avoir l'impression d'entrer dans un club.

# Philosophie Finale

Chaque écran doit répondre à une question :

"Est-ce que cela ressemble davantage à un club privé de passionnés de cinéma qu'à une application ?"

Si la réponse est non, supprimer l'élément.

La sensation recherchée n'est pas :

"Je gère un événement."

La sensation recherchée est :

"Je participe à une expérience collective autour d'un film."
